# Sprigly Landing Project

A full-stack animated landing page for Sprigly, a smart countertop herb garden.

## Run locally

```powershell
npm install
npm start
```

Then open:

```text
http://localhost:3000
```

## What is included

- `server.js` - Node HTTP server and waitlist API
- `public/` - Landing page, CSS, GSAP motion, and product asset
- `public/vendor/` - Local GSAP and ScrollTrigger browser files
- `data/waitlist.json` - Local JSON waitlist store
- `work/create_product_asset.ps1` - Optional script to regenerate the product PNG
