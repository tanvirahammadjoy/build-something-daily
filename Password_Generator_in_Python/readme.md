# How to Run

Save as password_generator.py
Run with: python password_generator.py

Learning Breakdown:

Variables: length, use*symbols, characters, etc.
Loops: while True for input validation + list comprehension with _
Random module: random.choice()
Functions: generate_password() and main()
String module: Easy access to letters, digits, punctuation

Enhancements You Can Try:

Add option for numbers only / letters only
Guarantee at least one of each character type
Save passwords to a file
Strength checker
